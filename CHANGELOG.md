# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Complete REST API client with 92+ methods
- WebSocket client for real-time streaming
- Pagination iterators with full parameter support
- Request options builder for per-request customization
- Retry logic with exponential backoff
- Structured logging system
- CMake package configuration for easy integration
- Support for all Massive API endpoints

### Changed
- Initial release

## [0.1.0] - 2024-11-21

### Added
- Initial implementation of massive-cpp
- Core infrastructure (HTTP transport, JSON parsing, configuration)
- REST API clients for all endpoints
- WebSocket client
- Complete feature and parameter parity with massive-python
- MIT License
- Comprehensive documentation
